Forex Trading App

The Forex Trading App is a simulated platform meticulously crafted to replicate the experience of forex trading. In the world of forex, participants buy and sell currencies with the expectation of value appreciation. Our app mirrors this process, utilizing a websockets API to provide real-time values for both GDP and USD currencies. Additionally, it offers user-friendly features, including registration, login, and validation functionalities for a comprehensive trading simulation experience.

Tech Stack: TypeScript, Node, React, Express, MongoDB, REST APIs, WebSockets API
